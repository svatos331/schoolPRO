# Запуск проекта

#### Для запуска проекта через yarn:
### `cd app/`
### `yarn start`
#
#### Для запуска проекта через npm:
### `cd app/`
### `npm run start`
#
#### Для запуска проекта через Docker:
### `cd app/`
### `yarn build`
### `cd ../`
### `docker-compose -f docker-compose.develop.yml up --build -d`

# Запуск линтера

#### Для сборки билда через yarn:
### `cd app/`
### `yarn lint`
